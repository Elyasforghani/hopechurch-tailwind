# Hope City Church — Tailwind Redesign

A modern, responsive redesign of a church website built with Tailwind CSS. Features a warm hero section, service times, "What to Expect" highlights, sermon library, member testimonials, and an embedded map.

![Hope City Church Screenshot](https://github.com/user-attachments/assets/71e5c5e3-cad5-4b8a-be89-5af461329488)

## 🔗 Live Demo

[View Live Site →](https://elyasforghani.github.io/hopechurch-tailwind)

## ✨ Features

- Responsive navigation with dropdown menus
- Hero section with call-to-action buttons
- Service times & embedded Google Map
- Sermon library with video thumbnails
- Member testimonials
- Footer with contact info & social links

## 🛠️ Built With

- Tailwind CSS
- HTML5

## 🚀 Getting Started

Clone the repo and open `index.html` in your browser — no build step required.

```bash
git clone https://github.com/Elyasforghani/hopechurch-tailwind.git
```
